Compare MAD Sheets Function

function compareMAD(setABeginning, setA, setBBeginning, setB) {
  if(setABeginning == "" || setBBeginning == "") {
    var result = "Please enter data in both sets.";
  } else {
    if(setA > setB) {
      var result = "Set B has less variability.";
    } else if(setA < setB) {
      var result = "Set A has less variability.";
    } else if(setA == setB) {
      var result = "Sets A and B have the same variability.";
    } else {
      var result = "Oops! Something went wrong.";
    }
  }
  return result;
}

Purpose

This function compares the MADs of two data sets.

Usage

COMPAREMAD(setABeginning, setA, setBBeginning, setB)

Arguments

  1. setABeginning - The first data point of set A
  2. setA - The MAD of set A (using AVEDEV)
  3. setBBeginning - The first data point of set B
  4. setB - The first data point of set B

Possible Results

  • "Set B has less variability."
  • "Set A has less variability."
  • "Sets A and B have the same variability."
  • "Oops! Something went wrong."
